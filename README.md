# 🏨 Hotel Management System

A Hotel Management System developed using **Java**, **JDBC**, **MySQL**, and the **MVC (Model-View-Controller)** architecture. This project is designed to simplify hotel operations by managing customer information, room bookings, and hotel records through a structured and user-friendly system.

---

## 📌 Features

- 👤 Customer Registration
- 🏨 Room Booking Management
- 📋 View Customer Details
- ✏️ Update Customer Information
- ❌ Delete Customer Records
- 🔍 Search Customer Records
- 💾 MySQL Database Integration
- 🔗 JDBC Connectivity
- 🏗️ MVC Architecture Implementation

---

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/khushbuthakur102005-cell/HotelManagementSystem1.git
```

### Steps to Run

1. Import the project into Eclipse.
2. Create the required MySQL database.
3. Update the database username and password in the configuration file.
4. Add the MySQL Connector JAR.
5. Run the application.
